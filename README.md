## 1. 개요

### 🔍 목표

- **한 줄 정리**: 유저가 영화 정보를 보고 의견을 남길 수 있는 웹 페이지 제작

  1. 입력한 평점을 기반으로 추천을 받을 수 있는 서비스

  2. 찜하기 기능을 통해 자신의 페이지에서 확인 할 수 있는 서비스
  3. 인기 영화, 인기 배우를 조회할 수 있는 서비스
